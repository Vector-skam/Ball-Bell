# Privacy Policy for ball-bals

This Privacy Policy describes how **ball-bals** (the "Project") handles data.

## 1. Information Collection and Use
**ball-bals** is a JavaScript-based project. We prioritize user privacy and follow these principles:
*   **No Data Collection:** The code operates entirely on the client side (in your browser) or within your local environment. We do not collect, store, or transmit any personal identification information to external servers.
*   **Local Storage:** If the project saves any settings or progress, it is stored locally on your device via `localStorage` or `cookies` and is not accessible by the developer.

## 2. Third-Party Services
This project does not include tracking scripts, analytics, or third-party advertising tools. Any external libraries used are for functional purposes only.

## 3. Security
The source code is open for audit. As an open-source project, you are encouraged to review the code to ensure transparency and data safety.

## 4. Changes to This Policy
I may update this Privacy Policy from time to time. Any changes will be posted in this file within the repository.
